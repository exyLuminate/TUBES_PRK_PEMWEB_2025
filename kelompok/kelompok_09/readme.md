## Ringkasan Proyek
Aplikasi web marketplace sosial untuk mendistribusikan makanan berlebih (sisa event/katering) dari donatur kepada mahasiswa di lingkungan kampus. Tujuannya mengurangi limbah pangan melalui sistem klaim berbasis kuota.

## Cara Instalasi
1. Import database `db_zerowaste.sql`.
2. Sesuaikan konfigurasi di `config/database.php`.
3. Buka `index.php` di browser.
